## Project Title

Home Automation via Bluetooth

## Objective 

To control home appliances such as lights, fans, and other electrical devices using a smartphone through Bluetooth communication.

## Components Required

- Arduino Uno
- HC-05 Bluetooth Module
- 4-Channel Relay Module
- Bulb/Lamp
- Fan (optional)
- Jumper Wires
- Breadboard
- Power Supply
- Android Smartphone
- 
## Working Principle

The HC-05 Bluetooth module receives commands from a smartphone.
Arduino processes the received commands.
Arduino turns relays ON/OFF according to the commands.
Relays control electrical appliances safely.

## Features

- Wireless control of appliances.
- Low cost and easy to build.
- Energy saving.
- User-friendly operation.
